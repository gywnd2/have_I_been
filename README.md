# 가봤을지도?
### 21-2학기 모바일 앱 개발 Term project
    Android Studio Arctic Fox(Java)
    SQLite
    --------------------------------------
    Java -> Kotlin
    SQLiteDatabase -> Room 으로 전환 중

## 1. 사진 탐색
- 내부 저장소 사진 탐색
## 2. 지오코딩
- 역지오코딩을 통한 좌표 -> 주소 체계 변환
## 3. 지도 API
- 네이버 지도 API를 사용하여 마커 표시
## 4. 기록 남기기
- 마커에 별점과 한마디, 자세한 코멘트 등 추억 남기기
